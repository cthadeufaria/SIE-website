BASIC TEXT FORMATTING

Let's start by basic text formatting commands.

This paragraph starts with regular font but includes bold and italic tags: this part of the text is in bold and this is in italic. From now on, we returned to regular text.

We can also mix this commands as follows: lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam urna turpis, accumsan vitae pellentesque eu, volutpat id lacus. Orci varius natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus. Integer orci justo, dictum a justo et, facilisis mollis lorem. Vestibulum ullamcorper aliquam magna id mollis.


Note that HTML ignores blank spaces and new lines as in this example: Lorem ipsum dolor sit amet, consectetur adipiscing elit. In sit amet elit in velit accumsan finibus. Pellentesque facilisis id orci vitae ultrices. Phasellus et velit.
